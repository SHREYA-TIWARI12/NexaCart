# Responsive E-Commerce Frontend

React.js, Vite, Tailwind CSS, Redux Toolkit, React Router, Axios-style API loading, React Hook Form, and Yup.

## Features

- Responsive product catalog with search, category filter, price filter, rating filter, and sorting.
- Product detail page with gallery-style product presentation.
- Persistent shopping cart with quantity controls and order summary.
- Wishlist management persisted in localStorage.
- Login/register UI with a demo JWT-like local session.
- Multi-step checkout with form validation.
- Mobile-first Tailwind CSS layout and reusable UI sections.
- API fetch from DummyJSON with local product fallback data.

## Run Locally

```bash
npm install
npm run dev
```

Open `http://127.0.0.1:5173/`.

## Verify

```bash
npm run build
npm run lint
```
